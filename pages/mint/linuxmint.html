O que é Linux Mint?

É uma distribuição Linux Irlandesa que foi concebida por Clément Lefèbvre e que é ativamente desenvolvida pela equipe Linux Mint Team e por sua comunidade. Baseado em Ubuntu (Distribuição baseada em Debian) e Debian (Baseado em Linux kernel).

O Mint tem 3 tipos de ambientes: Cinnamon (Versão mais popular e mais atrativa visualmente), MATE (Versão mais estável que a Cinnamon) e Xfce (Tem interface mais simples, indicado para computadores mais simples e de baixo desempenho).

Como fazer um pendrive bootável?

Vamos utilizar o programa “Ventoy”, que permite transformar o pendrive numa unidade com boot, suportando arquivos ISO. Para baixar o Ventoy primeiro vamos entrar no site oficial. Link

Após entrar no site vamos baixar o programa clicando na opção: ventoy-1.0.97-wind

Após clicar, o usuário será direcionado para o site SourceForge, site que os desenvolvedores utilizam para guardar os programas, temos que clicar no ventoy-10.0.97-windows.zip

Após clicar nele, o usuário vai ser redirecionado para o download automático. Após o tempo acabar irá mostrar a tela do explorador de arquivos para o usuário escolher onde vai salvar

Após escolher o local e clicar em salvar. Apenas devemos clicar com o botão direito e extrair o arquivo. Lembrando que: Se tiver um software compactador e descompactador de dados (Exemplo: Winrar e 7-zip) tem como extrair da mesma maneira. No tutorial está sendo usado o nativo do Windows.

Após clicar em “Extrair Tudo…” devemos escolher onde queremos extrair, nesse caso, vamos extrair tudo na mesma pasta. Após clicar em “Extrair”, começará a carregar e vai aparecer a mesma pasta, porém, extraída.

Já na pasta vamos ter todos os arquivos do programa, para instalá-lo precisamos abrir o “Ventoy2DIsk”, pois, se trata do programa. Quando você clicar no arquivo, o windows perguntará se você deseja mesmo abrir, para fins de segurança isso é um procedimento padrão, devemos clicar em SIM pois o ventoy é código aberto e não apresenta malware.

Após ter clicado em abrir, irá mostrar a interface do software. Com um compartimento chamado “device” que significa o pendrive que está espetado no PC. Detalhe: é importante abrir o programa com o pendrive espetado. Caso espete o pendrive depois, clique no ícone verde ao lado do device. Após aparecer seu pendrive clique em “install” e ele fará a dupla verificação se você realmente deseja formatar o pendrive, pois, apagará todos os dados contidos no pendrive.

Após clicar ‘’Sim’’ em todos os avisos, irá começar a carregar. Quando finalizar, a mensagem ao lado irá aparecer. Essa mensagem informa que o Ventoy foi instalado com sucesso no pendrive!
Detalhe: ele trocará o nome do pendrive por “Ventoy”.

Você deve selecionar a imagem baixada com Ctrl + X (recortar o arquivo que está salvo no computador para o pendrive) e clicar no pendrive e colar com o Ctrl + V. Após carregar, pronto, a ISO está no pendrive pronta para dar o boot no computador. Lembrando que é possível colocar várias ISO’s no mesmo pendrive e escolher qual dar o boot. Como mostra na imagem. Como dar boot no computador

  • Com o pendrive pronto e espetado no gabinete, vamos dar o boot no computador! Boot é iniciar o computador, apenas temos um boot bem sucedido quando temos o SO (sistema operacional) completo, nesse caso estamos utilizando o termo para formatar o computador num novo SO. Siga o passo a passo:
  • Desligue o computador;
  • Ligue novamente o computador;
  • Assim que ligar, aperte o botão “delete” do seu teclado para conseguir entrar na BIOS do seu computador. O botão de iniciar BIOS (Conjunto de instruções que permite que o processador trabalhe junto com os periféricos) pode variar de acordo com cada computador, normalmente é o “delete” do teclado. Caso não funcione, tente as teclas: Esc, F1, F2, F10 e F12 (não em sequência, tente uma de cada vez);
  • Na tela de BIOS, vá na parte de Boot no canto superior;
  • Quando estiver na aba de Boot coloque o seu pendrive como o “boot 1”. Ele deve ser o primeiro a iniciar junto com o windows;
  • Após selecionar o pendrive como primeiro, vá na última aba que tem “Save and exit” e clique na opção “Save and exit”;
  • Seu computador irá reiniciar e vai abrir a tela inicial do Ventoy e assim você clica com “enter” e vai começar a formatação!

Vantagens

  • Comunidade ativa

A comunidade do Linux Mint é extremamente ativa e engajada em ajudar com dificuldades de novos usuários na utilização do sistema. Com fóruns e suporte online, se destaca como uma comunidade acolhedora com usuário novo ou avançado. O Debian, porém, não tem uma comunidade tão acessível em relação ao Mint.

  • Interface

A interface do Linux Mint é conhecida por ser amigável e muito intuitiva, fazendo assim novos usuários não terem problemas com adaptação logo no início. Já o Debian tem uma interface muito mais técnica considerada até difícil para usuários avançados.

  • Drivers pré-instalados

O Mint tem drivers, como os de WI-FI e vídeo, já pré-instalados dependendo do seu hardware. Deixando assim muito mais fácil a configuração inicial para um usuário iniciante. Em diferença ao Debian que, precisa de um esforço técnico para a instalação dos mesmos drives.

Desvantagens

  • Consome mais recursos

O Mint, por ter questões de ter interface amigável para usuário iniciante e toda personalização estética, consome muito do hardware e de recursos do usuário. Em diferença ao Debian, que tem interface mais simples, porém, muito mais leve.

  • Atualizações não frequentes

O Mint utiliza de versões estáveis de Ubuntu, fazendo assim não ter tantas atualizações para o usuário e, consequentemente, o sistema pode “estagnar” numa configuração específica não tão indicada no momento. Já o Debian tem suas atualizações “rolling release” que trazem atualizações frequentes ao sistema.

  • Não tão reconhecido em corporações

O Mint não é tão utilizado em empresas tecnológicas quanto o Debian, que tem mais destaque nesse aspecto por ter recursos adicionais à proteção.
